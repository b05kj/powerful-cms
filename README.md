# powerful-cms

Learning project for powerful-cms

## Description
This project is focused on creating a powerful content management system (CMS) using Python.

## Features
- User authentication
- Content creation and management
- RESTful API for content access

## Tech Stack
- Python
- Flask

## Installation
To set up the project, run:
```bash
pip install -r requirements.txt
```

## Usage
To run the application, use:
```bash
python app.py
```

## Future Plans
- Add more features such as role-based access control
